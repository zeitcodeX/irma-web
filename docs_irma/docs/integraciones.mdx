Irma está diseñada para ser el centro de tus comunicaciones, por eso se integra de forma nativa con las plataformas más importantes para ventas y atención al cliente.

### WhatsApp
Es nuestro canal principal. La integración con la API de WhatsApp Business permite que Irma converse con tus clientes de forma fluida y directa en la plataforma de mensajería más usada del mundo, manteniendo el historial y permitiendo la intervención de un agente humano en cualquier momento.

### Shopify y Tiendanube
Para los e-commerce, la conexión con el catálogo es vital. Irma se integra con Shopify y Tienda Nube para:
- **Acceder a tu catálogo de productos en tiempo real:** Puede responder consultas sobre precios, stock, variantes (talles, colores) y especificaciones.
- **Hacer recomendaciones inteligentes:** Sugiérele productos complementarios a tus clientes.
- **Entender el estado de un pedido:** (Próximamente) Podrá consultar el estado de un envío directamente desde el chat.

### Sitios Web Propietarios
¿Tenés una web o un sistema a medida? No hay problema. Irma puede integrarse en cualquier página, consultanos por las opciones que tenemos para vos según la arquitectura de tu sitio.
